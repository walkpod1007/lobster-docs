# 🦞 龍蝦系統文件中心

> 維護者：住家德瑪 | 協作：辦公室小蝦 | 顧問：web Opus

德瑪有 GitHub 工具可直接更新這裡。你把 Opus 的輸出帶給德瑪，德瑪存進來，所有人都能查閱。

## 資料夾結構

| 資料夾 | 說明 |
|--------|------|
| `collab/` | Opus 出的工單、架構規劃（COLLAB-WO/ARCH/SPEC/SOP） |
| `arch/` | 系統架構文件（長期有效） |
| `sop/` | 標準操作流程 |
| `review/` | 驗證報告、審查結果 |

## 如何使用

1. 你和 Opus 討論出文件
2. 把連結或內容帶給德瑪：「幫我存進 GitHub」
3. 德瑪 commit 進對應資料夾
4. 用這個 repo 的連結分享給任何人
